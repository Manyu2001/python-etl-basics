# Python ETL Basics

Mini ETL pipeline built during Week 1 of my Data Engineer upskilling plan.

## What it does
- Reads Olist customers CSV using pandas
- Cleans city/state formatting
- Adds customer region column
- Runs data quality checks
- Saves cleaned output to CSV
- Logs every step using Python logging

## How to run
pip install pandas
python mini_etl.py

## Tech used
Python, pandas, logging, os
